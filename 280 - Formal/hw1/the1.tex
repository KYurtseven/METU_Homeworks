\subsection*{a.}

If a language $L$ is regular, then there exists a number $n \geq 1$ s.t. every string $uwv$ in $L$ with $|w| \geq n$ can be written in the form $ w = xyz$ s.t. $|y| \geq 1$ and $|xy| \leq n$ and $xy^iz \in L$ for $i \geq 0$.
